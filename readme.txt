The Easiest Way Manage Your Photo Albums in Wordpress

== Description ==

Easily Manage Your Photo Albums using jQuery and prettyPhoto. No fuss to use, or setup. The simple hierarchy system means zero difficulty in creating and managing image categories.

== Installation ==

1. Upload smartFolio to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start creating albums
